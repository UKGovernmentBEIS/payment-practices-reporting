Publish a report for @companyName

If you want a colleague who has a Companies House authentication code to publish the report, follow these steps.

  1. Give them the report data, approved by a director, or a designated member if the business is a Limited Liability Partnership.
  2. Make sure you also give them the deadline to publish the report.
  3. They will need to use their Companies House authentication code to publish the report.
